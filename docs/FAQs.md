# Frequently Asked Questions (FAQs)

#### **1. Question**: I need to test and adjust this solution accelerator to work with my own company's data. What approach should I take?

**Answer**: We recommend deploying and configuring this solution first to understand how the sample data is used and how information flows through the solution. Then you can customize different components to fit your own needs.

#### **2. Question**: Can I deploy a sub-component of the solution accelerator?

**Answer**: All components in the solution need to work together as an integrated business solution. Our deployment guide includes instructions to deploy all components in the right sequence. For example, the Microsoft 365 Copilot Agent uses the Foundry Chat Agent and Fabric Ontology Data Agent, therefore, it will be deployed last after Foundry and Fabric Components have been successfully deployed.
