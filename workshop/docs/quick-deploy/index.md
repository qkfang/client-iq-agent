# Quick Deploy Overview

This Quick Deploy guide for the Build faster with Solution Accelerators - Foundry IQ + Fabric IQ workshop offers two deployment options. Please select the deployment path that best matches your requirements. Detailed step-by-step instructions for each option are provided in the following sections.


## 1. Microsoft Foundry + Microsoft Fabric Deployment
- **Recommended for full functionality**
- Deploys both the Microsoft Foundry and Microsoft Fabric components. Follow the deployment steps [here](./deployment-guide-optionA.md). 

## 2. Azure Only Deployment
- Deploys only the Microsoft Foundry components and Azure SQL Database. Follow the deployment steps [here](./deployment-guide-optionB.md).
